# 오피스아워 보고서 시스템 테스트 가이드

## 🔐 테스트 계정

### 1. 관리자 계정
```
이메일: admin@mysc.co.kr
비밀번호: (아무거나 입력)
권한: 모든 사업 접근, 전체 보고서 조회
```

### 2. 컨설턴트 계정
```
이메일: consultant@mysc.co.kr
비밀번호: (아무거나 입력)
담당 사업: 농식품(p1), 경기도(p3)
권한: 담당 사업 보고서만 조회
```

### 3. 실무진 계정
```
이메일: staff@mysc.co.kr
비밀번호: (아무거나 입력)
담당 사업: 해양수산(p2), 환경(p4)
권한: 담당 사업 보고서만 조회
```

### 4. 일반 사용자 계정
```
이메일: user@example.com (또는 아무 이메일)
비밀번호: (아무거나 입력)
권한: 일반 사용자 기능만
```

## 📝 보고서 작성 팝업 테스트 방법

### 방법 1: 완료된 세션 확인 (즉시 테스트)

현재 데이터에는 **과거에 완료된 세션**들이 있습니다:

1. **컨설턴트 계정으로 로그인**
   ```
   consultant@mysc.co.kr
   ```

2. 로그인하면 **자동으로 보고서 작성 팝업**이 뜹니다!
   - 완료된 세션 중 보고서가 없는 첫 번째 세션
   - 예: 2026년 1월 15일 완료된 "임팩트 측정" 세션

3. 팝업이 안 뜨면:
   - 로그아웃 → 다시 로그인
   - 또는 페이지 새로고침 (F5)

### 방법 2: 브라우저 콘솔에서 강제로 띄우기

1. 브라우저에서 **F12** (개발자 도구 열기)
2. **Console** 탭으로 이동
3. 다음 명령어 입력:
   ```javascript
   // 완료된 세션 목록 확인
   console.log("완료된 세션:", localStorage.getItem("mysc-applications"));
   ```

### 방법 3: Mock 데이터 수정 (실시간 테스트)

현재 시간으로 완료 세션을 만들려면:

1. `/src/app/lib/data.ts` 파일 열기
2. `app4` (또는 다른 completed 상태 신청) 찾기
3. `scheduledDate`와 `scheduledTime`을 현재 시간으로 변경:

```typescript
{
  id: "app4",
  type: "regular",
  status: "completed",
  // ... 다른 필드들
  scheduledDate: "2026-02-06", // 오늘 날짜로
  scheduledTime: "15:00", // 현재 시간 - 2시간으로
  duration: 2,
  // ...
}
```

4. 저장 후 **컨설턴트 계정**으로 로그인하면 팝업이 뜹니다!

## 🎯 보고서 시스템 전체 테스트 플로우

### 1단계: 로그인 및 자동 팝업 확인

```
1. consultant@mysc.co.kr 로그인
2. ✅ 보고서 작성 팝업 자동으로 표시됨
```

### 2단계: 보고서 작성

팝업에서 다음 정보 입력:

- **일시**: 자동으로 세션 일시 표시
- **장소**: 온라인/오프라인 선택
  - 오프라인 시 상세 주소 입력
- **주제**: 논의한 주요 주제 (필수)
- **참석자**: 참석자 이름 (쉼표로 구분)
- **내용**: 세션에서 논의한 내용 (필수)
- **팔로업**: 후속 조치 사항
- **사진**: 파일 첨부 (선택사항)
- **만족도**: 1-5점 선택 (필수)

### 3단계: 제출 옵션

- **"제출" 버튼**: 즉시 보고서 제출
- **"나중에 작성" 버튼**: 팝업 닫기 (미작성 보고서로 추적됨)

### 4단계: 미작성 보고서 추적

1. 좌측 사이드바에서 **"미작성 보고서"** 메뉴 클릭
2. 확인 가능한 정보:
   - 전체 미작성 보고서 수
   - 기한 초과 보고서 (3일 이상)
   - 사업별 통계
   - 컨설턴트별 통계
3. **"보고서 작성"** 버튼 클릭 → 작성 팝업 다시 열림

## 🐛 문제 해결

### 팝업이 안 뜰 때

1. **로그아웃 → 다시 로그인**
2. 브라우저 콘솔(F12) 확인:
   ```javascript
   // 완료된 세션 확인
   console.log("완료 세션:", applications.filter(a => a.status === "completed"));
   ```

### "미작성 보고서" 메뉴가 안 보일 때

- 일반 사용자 계정으로 로그인했는지 확인
- 컨설턴트/관리자/실무진 계정으로만 메뉴 표시됨

### 보고서 제출 후 확인

1. "미작성 보고서" 메뉴로 이동
2. 제출한 세션이 목록에서 사라졌는지 확인
3. 통계 숫자가 감소했는지 확인
